그럼 서버가 잘 실행된다

     "server": "json-server --watch db.json --port 3001"
서버 실행을 위해서 이런식로 해야하지만 너무 길어서 번거롭다
그걸 위해서 package.json에 추가하면 npm run server로 실행 가능함
